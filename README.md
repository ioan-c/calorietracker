# Calorie Tracker by MJ

### Prerequisites
* Docker
* Java 21

### Deploy the database:
* run ```docker-compose up``` command in project root

### Maven clean install:
* run in project root
* ```mvn clean install```
* or
* ```./mvnw clean install```

### Start the application:
* run in project root
* ```mvn spring-boot:run```
* or
* ```./mvnw spring-boot:run```
